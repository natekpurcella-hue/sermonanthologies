# Sermon Anthologies

A Markdown sermon corpus with tooling for retrieval, cleanup, voice generation,
YouTube drafts, short-form clips, and anthology/audiobook packaging.

## Key Areas

- `Charles Spurgeon/`, `Finney/`, `George Muller/`, `Wesley/`, `AW Tozer/`,
  and `Ravenhill/`: author sermon collections and indexes.
- `pipeline/`: content pipeline design, metadata contracts, and automation
  tools.
- `catalog/`: generated sermon catalog and scan reports.
- `Anthologies/`: curated text collections.
- `KAGGLE_PIPELINE.md` and `fish_speech_sermon.ipynb`: current Fish Speech GPU
  generation workflow.
